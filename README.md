囲み文字ジェネレータ

https://kakomimojigen.netlify.app/

テキストボックスに入力された文字に囲み文字をくっつけます！

## 例

入力
```
こんにちは、世界！
```

出力
```
こ⃝ん⃝に⃝ち⃝は⃝、⃝世⃝界⃝！⃝
```
